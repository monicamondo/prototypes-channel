Como usar os repositórios

Concentramos todos os acessos referente às entregas de design em um só lugar. Aqui você encontra o link para protótipo navegável, e o código HTML/CSS, que pode auxiliar na etapa de implementação.

  1. Na tabela, clique em Ver repositório, e acesse o diretório no GitHub com o código fonte das interfaces projetadas.
  2. O protótipo possui um componente novo? Você pode inspecioná-lo no StoryBook do Design System, lá projetamos todos os status e comportamento desejados para esse componente.
